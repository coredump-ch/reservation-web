# Ultimaker Reservation Tool

The web interface for the [Reservation API](https://github.com/coredump-ch/reservation-api),
written using [Svelte](https://svelte.dev/).

## Dev server

Set up configuration:

    cp .env.example .env

Run dev server:

    npm run dev

## Build production bundle

    npm run build
